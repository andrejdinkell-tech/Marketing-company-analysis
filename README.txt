Анализ эффективности маркетинговой кампании

Автор: Андрей Динкель  

Описание проекта
Цель проекта — оценить эффективность проведённых маркетинговых кампаний и выявить факторы, влияющие на продажи.  
Для этого были использованы данные о покупках клиентов и их социально-демографических характеристиках.

Постановка задачи
1. Проверить гипотезу:
   -H0: Кампания не изменила конверсию и средний чек.
   -H1: Кампания увеличила конверсию и средний чек.
2. Провести сегментацию клиентов для поиска целевых групп.
3. Построить модель прогнозирования склонности клиента к покупке.

Этапы работы
- Подготовка и очистка данных.
- Проведение A/B-теста.
- Кластеризация пользователей (KMeans).
- Анализ поведения кластеров и формирование маркетинговых рекомендаций.
- Построение модели склонности (Random Forest).
- Прогноз для города 1188.

Результаты
- Email-скидки увеличили конверсию и средний чек.
- Кластеризация выделила несколько сегментов с различным поведением.
- Наиболее перспективный сегмент: мужчины ~41 года, интересующиеся транспортом и мебелью.
- Random Forest достиг Accuracy = 0.63, F1 = 0.58.
- Прогноз показал, что 30–40% клиентов города 1188 имеют высокую вероятность покупки.

Выводы
- Кампания принесла положительный эффект (рост конверсии +1.9 п.п., рост ARPU +1.68%).
- Рекомендуется таргетировать кластер с наибольшей средней стоимостью покупок.
- Скидки эффективнее работают для «среднего сегмента», для премиум-клиентов важнее бренд.

Реализация в Python:

Используемые библиотеки:
- pandas / numpy — работа с данными
- matplotlib / seaborn — визуализация
- scikit-learn — A/B-тест, кластеризация, Random Forest
- scipy — статистические тесты

Содержимое ноутбука
1. Загрузка и предобработка данных 
   Объединение таблиц, очистка, работа с пропусками.

2. A/B-тест  
   Сравнение тестовой и контрольной групп.  
   Проверка статистической значимости различий.

3. Сегментация (KMeans)
   Разделение клиентов на группы по демографическим и поведенческим признакам.  
   Визуализация результатов.

4. Модель склонности к покупке (Random Forest) 
   Обучение, кросс-валидация, расчёт метрик (Accuracy, F1, ROC-AUC).

5. Прогноз для города 1188 
   Оценка доли клиентов с высокой вероятностью покупки.
